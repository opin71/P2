Unit 2, Outcome 1

1.7 Describe which are the appropriate filetypes for websites

Code File Formats for sites:
There are many file types that can be used for websites development and it depends on what you will use your website for, whether you will store data and have search functions which in this case you may also use SQL and Python or whether it is a simple landing page with video which you may just use html, css and javascript.

Media File Formats for sites:

Image File Formats
Jpg, Png and Gif all can be optimised and compressed and will still maintain lossless and Bmp as well as Tiff are often used for Print and occassionally for vector rendering.

Video File formats
Streaming video content from a third party who has robust global server coverage would be better such as Youtube, AWS, Vimeo, Wistia and others than uploading it on your own web hosting plan which may have a traffic limit and not cover all data centres globally which could mean some users may experience slowness in loading and streaming in countries.
In terms of video file content mp4 is a standard and easy to work with across all platforms. Though mov, flv, avi, mkv and other formats can be used depending on what the content is used for.

Audio File Formats
Mp3, Wav, Flac and others can be used for the web but Flac has the most lossless even though the files are much bigger that Mp3. When choosing an audio file format it depends on your audience and what the content is about.
